Portfolio

EcoTrend: Sustainable Living Dashboard

EcoTrend is an innovative web application designed to promote sustainable living by providing users with personalized eco-friendly tips and resources. Developed using React for a dynamic front-end experience and Node.js for a robust back-end, the dashboard integrates with various APIs to offer real-time updates on environmental news, local recycling guidelines, and green product recommendations.

TravelBuddy: Personalized Travel Itinerary Generator

TravelBuddy is a comprehensive travel itinerary generator built to simplify the planning process for globetrotters. Utilizing a combination of Vue.js for the front-end and Python Flask for the back-end, this application allows users to input their travel preferences, including destination, interests, and budget. The intelligent algorithm then generates a customized itinerary with suggested activities, accommodation options, and dining experiences.
